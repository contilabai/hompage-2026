# ISafePlanner: BIM 기반 동적 공정 및 안전 예측 스케줄러 (Plan Module)

철저한 데이터 기반 사전 계획으로 아차사고와 중대재해 가능성을 원천 차단합니다.

**모듈 색상:** 초록 (#14532d → #16a34a)  
**상품 페이지:** /isafe-planner

---

## 핵심 기능

### 1. 도면 자동 3D 전환 컨버터
2차원 dxf 도면이나 작업 WBS를 입력받아 업계 표준인 3차원 IFC 모델로 자동 전환하는 엔진 탑재

### 2. WBS 연동 가변 3D 모델 분할
공정 스케줄에 맞춰 벽체, 기둥, 슬래브 등 구조물 메쉬를 실시간 분할하여 시기별 공정 시각화

### 3. 공정별 동적 위험도 평가 가이드
자체 내장된 위험 요소 데이터베이스와 기상청 날씨 예측 API를 연동하여 가변적인 위험 지수 정량화 산출

### 4. 지능형 가변 카메라 배치 시뮬레이터
현재 공정률 및 구조물 사각지대를 가상 3D 공간에서 분석하여 최적의 이동형 관제 카메라 위치 자동 추천

---

## 모듈 연동
- ISafeMeta: ISafePlanner가 빌드한 현장 3D 모델을 ISafeMeta에 이식하여 가상 훈련 환경 구성
- ISafeGuard: 카메라 위치 추천 결과를 ISafeGuard에 전달하여 최적 관제 배치 구현
- ISafeChain: 위험도 분석 데이터를 ISafeChain 피드백 루프로 전달하여 위험도 가중치 자동 재조정

---

## 판매 포인트
- 종이 도면 → 3D BIM 모델 자동 전환으로 BIM 전문가 불필요
- 기상 API 연동으로 날씨 변화에 따른 위험도 실시간 갱신
- AI가 추천하는 최적 CCTV 위치로 사각지대 최소화
