# iSafe-Guard 인공지능 모델 패키지 구성 및 판매 전략 가이드

이 문서는 iSafe-Guard 솔루션의 핵심인 인공지능 디텍터 모델들을 객체 인식 난이도와 현장 적용 범용성을 기준으로 분류하고, 이를 3단계 판매 패키지로 구조화한 가이드라인입니다.

출처: 사용자가 제공한 116개 AI 디텍터 리스트 데이터 분석 기반

---

## 1. 패키지 구성 전략 개요

iSafe-Guard의 AI 모델들은 단순 객체 인식부터 복합적인 행동 분석, 현장 특화 환경까지 다양한 스펙트럼을 가집니다. 고객의 예산과 현장 요구사항에 맞춰 유연하게 대응할 수 있도록, 모델들을 3개의 티어(Tier)로 나누어 패키징합니다.

---

## 2. iSafe-Guard Basic (기본 패키지)

### 2.1 패키지 컨셉
- 범용성 최우선: 어느 현장에서나 공통으로 요구되는 필수 안전 관제 모델
- 빠른 도입: 단일 객체 인식이 비교적 명확하여 튜닝 없이 즉각적인 현장 적용이 가능한 모델군

### 2.2 포함 모델 리스트 (주요 디텍터)
- 개인 보호구 감지 모델: 
  - 공정: 외부비계, 사다리, 크레인, 철근, 터파기 등 대부분 공정
  - 감지 대상: 안전모 착용 여부 실시간 파악
- 필수 체결 장비 감지 모델: 
  - 공정: 외부비계, 철골 작업
  - 감지 대상: 안전고리 체결 여부 확인
- 필수 안전 인력 감지 모델: 
  - 공정: 고소작업대, 이동식/타워크레인, 터파기, 지게차 등 장비 운용
  - 감지 대상: 신호수 정상 배치 여부 확인
- 장비 및 근로자 충돌 방지 모델: 
  - 공정: 터파기, 백호, 지게차 작업
  - 감지 대상: 장비와 작업자 간의 안전거리 최소 유지 범위 파악
- 현장 기본 안전 수칙 모델: 
  - 공정: 물류 및 화기 작업
  - 감지 대상: 지게차 운행 제한속도 준수, 원형톱 절단 구간 소화기 비치 여부 확인

## 3. iSafe-Guard Pro (기본 + 특화 패키지)

### 3.1 패키지 컨셉
- 심층 분석 및 룰 연동: Basic 패키지의 모든 기능을 포함하며, 특정 공정의 전문적인 자재 인식과 시계열 행동 분석이 결합된 고도화 모델
- 복합 상호작용 판별: 단일 객체가 아닌 객체 간의 복잡한 상호작용과 룰 위반을 잡아내는 기능 포함

### 3.2 포함 모델 리스트 (특화 디텍터)
- 중장비 디테일 감지 모델: 
  - 사다리, 이동식 비계, 고소작업대, 크레인, 펌프카의 전도방지장치 및 침하방지 장치 정상 설치 여부
- 복합 행동 및 상태 인식 모델: 
  - 비계를 타고 오르거나 몸을 밖으로 내미는 이상 행동 감지
  - 부적절한 자재를 밟고 작업하는 불안전 상태 파악
- 고위험 고소 작업 특화 모델: 
  - 고소작업대 3인 이상 탑승 및 탑승 상태로 주행 이동하는 행위 감지
  - 차량 탑재형 고소작업대의 고임목 3단 이상 위험 사용 판단
- 크레인 및 양중 특화 모델: 
  - 크레인 자재 양중 시 하방 수직 낙하 구역 내 작업자 진입 통제
  - 타워크레인 양중 시 30cm 높이에서 3초간 일시 정지하는 필수 안전 룰 준수 여부
- 가설 구조물 및 환경 안전 모델: 
  - 외부비계 상하부 동시 작업 통제, 철근캡 설치 유무, 동바리 수직도 적정 여부 확인
  - 개구부 덮개 및 안전난간 설치 여부, 절단 작업 시 불티비산방지망 정상 사용 여부

## 4. iSafe-Guard Custom (맞춤 제작 패키지)

### 4.1 패키지 컨셉
- 엔터프라이즈 전용: 현장마다 규격이 다른 특수 장비나 특정 건설사 본사 고유의 깐깐한 안전 내규를 시스템에 반영하기 위한 맞춤형 패키지
- 밀착 관리: 현장 실측 데이터를 바탕으로 바닥부터 새롭게 학습 및 지속적인 파인 튜닝(Fine-Tuning) 지원

### 4.2 주요 제공 서비스
- 커스텀 장비 및 자재 데이터 세트 학습: 
  - 기성 모델로 탐지가 어려운 현장 특수 시공 자재, 자체 제작한 인양박스, 규격 외 특수 중장비 등의 정밀 학습 및 객체 추적
- 현장 전용 복합 룰 베이스 구축: 
  - (예시) 특정 통제 구역 내에 지정된 장비 2대와 근로자 3명 이상이 동시에 얽힐 때만 알람을 울리는 등의 복잡한 다중 조건 맞춤형 로직 생성
- 밀착형 오탐지 보정 파이프라인 연동: 
  - 해당 현장 특유의 환경 변수(역광, 특수 조명, 특정 분진 발생량, 특이한 카메라 화각 등)에 완벽히 동기화
  - 전담 엔지니어가 지속적으로 오탐지 데이터를 필터링하고 백엔드 모델 재학습을 통해 가중치 업데이트 관리 유지
